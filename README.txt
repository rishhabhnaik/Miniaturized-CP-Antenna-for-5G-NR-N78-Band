All files are typical HFSS models that can be opened and analyzed in AnsysEM
version 2020 R2 or later

"LP original.aedt"
Linearly polarized antenna model using the exact dimensions given in the
original paper

"Single Feed original.aedt"
Circularly polarized antenna model with a single feed using the exact
dimensions given in the original paper

"Dual Feed original.aedt"
Circularly polarized antenna model with two feeds using the exact
dimensions given in the original paper

"LP ours.aedt"
Linearly polarized antenna model we modified for the presentation.

"Single Feed 3p5GHz 0p26BW 3p8AR.aedt"
Circularly polarized antenna model with a single feed using our modified
dimensions
